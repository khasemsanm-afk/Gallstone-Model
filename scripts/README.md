# Scripts Documentation

โฟลเดอร์นี้รวบรวมสคริปต์ภาษา Python ที่ใช้สำหรับกระบวนการพัฒนา AI Model (MLOps Lifecycle) ตั้งแต่การเตรียมข้อมูล การฝึกสอน การแปลงไฟล์ จนถึงการประเมินผลความแม่นยำ

ไฟล์ต่างๆ ถูกตั้งชื่อเรียงตามลำดับขั้นตอนการทำงาน (01 ถึง 06) เพื่อให้ง่ายต่อการทำความเข้าใจและนำไปใช้งานต่อในอนาคต

---

## รายละเอียดของแต่ละสคริปต์

### 1. `01_generate_jsonl.py` (สคริปต์เตรียมข้อมูลสอน)
- **หน้าที่:** แปลงข้อมูลดิบจากไฟล์ Excel (ที่ได้รับการ Label จากแพทย์) ให้กลายเป็นรูปแบบ ChatML (JSONL) ที่ AI เข้าใจได้
- **กระบวนการ:** 
  - สกัดเฉพาะข้อความ (Evidence / Impression) ที่เกี่ยวข้องกับถุงน้ำดี
  - จัดรูปแบบ Prompt (คำสั่ง) ขาเข้า และ JSON ขาออก
  - สุ่มแบ่งชุดข้อมูลออกเป็น **Training Set (85%)** และ **Validation Set (15%)** สำหรับใช้ทดสอบ

### 2. `02_train_unsloth.py` (สคริปต์ฝึกสอนโมเดล)
- **หน้าที่:** ทำการ Fine-tune AI ด้วยเทคนิค QLoRA
- **กระบวนการ:** 
  - โหลด Base Model (`google/medgemma-1.5-4b-it`) และชุดข้อมูล Training Set
  - ใช้ไลบรารี `unsloth` ในการบีบอัดการใช้หน่วยความจำ (VRAM) ทำให้สามารถรันบนชิปกราฟิกระดับกลาง (เช่น NVIDIA L4) ได้
  - บันทึกน้ำหนัก (Weights/Adapter) ที่เรียนรู้เสร็จแล้วลงในโฟลเดอร์ที่กำหนด

### 3. `05_export_to_ollama.py` (สคริปต์แปลงไฟล์เป็น GGUF)
- **หน้าที่:** ผสมโมเดลหลักเข้ากับน้ำหนักที่ Fine-tune มา และแปลงเป็นรูปแบบที่เหมาะกับการรันบน CPU
- **กระบวนการ:**
  - ทำการ Merge LoRA Adapter เข้ากับ Base Model
  - Quantize โมเดลให้เล็กลงในระดับ 4-bit (Q4_K_M)
  - ส่งออกเป็นไฟล์ `.gguf` ซึ่งเป็นไฟล์ที่สามารถนำไปใช้งาน (Deploy) ผ่านระบบ Ollama ได้ทันที

### 4. `06_evaluate_ollama.py` (สคริปต์ประเมินผลผ่าน Ollama)
- **หน้าที่:** ตรวจข้อสอบโมเดลอย่างรวดเร็วและสร้างรายงานทางสถิติ
- **กระบวนการ:**
  - โหลดชุดข้อมูลข้อสอบ Validation Set
  - ยิงคำถามผ่าน HTTP POST ไปยัง Ollama ที่กำลังทำงานอยู่ในเครื่อง Local (ใช้เวลาประมวลผลสั้นเพียง 2-5 วินาทีต่อคำถาม)
  - นำผลลัพธ์ JSON มาเปรียบเทียบกับคำตอบจริง (Ground Truth)
  - คำนวณค่าสถิติ (Accuracy, Sensitivity, Specificity)
  - บันทึกข้อที่โมเดลทำนายผิดลงในไฟล์ Excel (`evaluation_errors.xlsx`) เพื่อให้ผู้เชี่ยวชาญวิเคราะห์ต่อไป
